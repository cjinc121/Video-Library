# BingeIt - https://bingeit.vercel.app/

## About
A Video Library app for streaming across various categories 

## Tech Stack
- ReactJs
- Redux Toolkit
- Backend - Mockbee

## Features 
- User Authentication
- Watch Later
- Liked
- Playlist Management
- History tracking
- Search Video
